## Reportar una Vulnerabilidad

Por favor, contacta directamente a los administradores a través de:
- **Email:** david.diz.oubina@udc.es (Asunto: SECURITY REPORT)

**Por favor, proporciona:**
- Descripción detallada del problema.
- Pasos para reproducirlo (un "PoC" o código de ejemplo es ideal).
- Impacto potencial (ej: "acceso a datos de usuario", "caída del sistema").

Nos tomamos la seguridad muy en serio y trabajaremos para abordar cualquier problema reportado lo antes posible.
